## Usage

```
cargo run <file-path>
# or
./elfpeach <file-path>
```

|  key  |  description  |
| ---- | ---- |
|  `q`  |  quit  |
|  `←/→`  |  change attribute  |
|  `↑/↓`  |  change section/segment/symbol  |
